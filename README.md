🚗🏎️ Super Trunfo de Carros Antigos

Aplicativo mobile desenvolvido com React Native + Expo, inspirado no clássico jogo Super Trunfo, focado em carros antigos brasileiros.

📱 Sobre o Projeto

O objetivo deste app é recriar a experiência do jogo Super Trunfo, onde cada carta representa um carro com suas características e atributos.

O usuário pode visualizar cartas com informações detalhadas e, futuramente, comparar veículos em batalhas.

🧠 Funcionalidades

✅ Exibição de cartas de carros
✅ Imagem do veículo
✅ Informações principais (marca, ano, origem)
✅ Descrição do carro
✅ Sistema de avaliação por estrelas ⭐
✅ Interface moderna usando React Native Paper
